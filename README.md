# Apache Libprocess

**this branch libprocess is part of mesos for libprocess lib only**

# Installation

```sh
cd mesos && ./bootstrap
export PYTHON=/bin/python2.7 # 需要python2支持
mkdir build && cd build && ../configure
make -j
# make check
make install # 安装到 
```
